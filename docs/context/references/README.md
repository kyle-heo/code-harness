## 파일 추가 기준

- 에이전트가 구현 중 반복 참조하는 외부 문서
- 공식 문서가 LLM 컨텍스트 창에 비해 과도하게 길어 핵심만 추출한 것
- 파일명은 반드시 `*-llms.txt` 형식 (용도 명시)

## 갱신 방법

stale 여부를 감지하면 Claude Code가 최신 문서를 재추출하여 갱신합니다.
