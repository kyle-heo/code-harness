## 심각도 레벨

| 심각도 | 정의 | 예시 |
|--------|------|------|
| **critical** | 핵심 워크플로우 차단, 데이터 손실, 앱 충돌 | 폼 제출 시 에러 페이지, 결제 흐름 중단, 확인 없이 데이터 삭제 |
| **high** | 주요 기능 손상 또는 사용 불가, 우회 방법 없음 | 검색이 잘못된 결과 반환, 파일 업로드 무음 실패, 인증 리다이렉트 루프 |
| **medium** | 기능은 동작하지만 눈에 띄는 문제, 우회 방법 있음 | 느린 페이지 로드(>5초), 폼 유효성 검사 누락, 모바일에서만 레이아웃 깨짐 |
| **low** | 미미한 시각적 또는 polish 이슈 | 바닥글 오타, 1px 정렬 이슈, 일관성 없는 hover 상태 |

## 카테고리

### 1. 시각/UI
- 레이아웃 깨짐 (요소 겹침, 텍스트 잘림, 가로 스크롤바)
- 깨지거나 누락된 이미지
- 잘못된 z-index (요소가 다른 것 뒤에 나타남)
- 폰트/색상 불일치
- 애니메이션 결함 (jank, 미완성 전환)
- 정렬 이슈 (그리드 벗어남, 불균등 간격)
- 다크 모드 / 테마 이슈

### 2. 기능
- 깨진 링크 (404, 잘못된 목적지)
- 죽은 버튼 (클릭해도 아무것도 안 됨)
- 폼 유효성 검사 (누락, 잘못됨, 우회됨)
- 잘못된 리다이렉트
- 상태 유지 안 됨 (새로 고침 시 데이터 손실, 뒤로 버튼)
- 경쟁 조건 (이중 제출, 오래된 데이터)
- 잘못된 또는 없는 검색 결과

### 3. UX
- 혼란스러운 내비게이션 (빵 부스러기 없음, 막다른 길)
- 로딩 인디케이터 누락 (무언가 일어나고 있다는 사용자 인식 없음)
- 느린 상호작용 (피드백 없이 >500ms)
- 불명확한 에러 메시지 ("문제가 발생했습니다" 세부 없음)
- 파괴적 작업 전 확인 없음
- 페이지 전반 일관성 없는 상호작용 패턴
- 막다른 길 (뒤로 갈 방법 없음, 다음 동작 없음)

### 4. 콘텐츠
- 오타 및 문법 오류
- 오래되거나 잘못된 텍스트
- 남아있는 placeholder / lorem ipsum 텍스트
- 잘린 텍스트 (말줄임표나 "더 보기" 없이 끊김)
- 버튼 또는 폼 필드의 잘못된 레이블
- 누락되거나 도움이 안 되는 빈 상태

### 5. 성능
- 느린 페이지 로드 (>3초)
- 버벅거리는 스크롤 (프레임 드랍)
- 레이아웃 이동 (로드 후 콘텐츠 점프)
- 과도한 네트워크 요청 (단일 페이지에서 >50)
- 최적화되지 않은 대형 이미지
- 블로킹 JavaScript (로드 중 페이지 무응답)

### 6. 콘솔/에러
- JavaScript 예외 (잡히지 않은 에러)
- 실패한 네트워크 요청 (4xx, 5xx)
- 사용 중단 경고 (다가올 중단)
- CORS 에러
- 혼합 콘텐츠 경고 (HTTPS 페이지의 HTTP 리소스)
- CSP 위반

### 7. 접근성
- 이미지에 alt 텍스트 누락
- 레이블 없는 폼 입력
- 키보드 내비게이션 깨짐 (요소에 탭 불가)
- 포커스 트랩 (모달이나 드롭다운에서 빠져나올 수 없음)
- 누락되거나 잘못된 ARIA 속성
- 불충분한 색상 대비
- 스크린 리더로 접근 불가 콘텐츠

## 페이지별 탐색 체크리스트

QA 세션 중 방문하는 각 페이지에 대해:

1. **시각 스캔** — 스크린샷 캡처. 레이아웃 이슈, 깨진 이미지, 정렬 확인
2. **인터랙티브 요소** — 모든 버튼, 링크, 컨트롤 클릭. 각각이 하는 일을 하는가?
3. **폼** — 채우고 제출. 빈 제출, 잘못된 데이터, 엣지 케이스 (긴 텍스트, 특수 문자) 테스트
4. **내비게이션** — 모든 진입/진출 경로 확인. 빵 부스러기, 뒤로 버튼, 딥 링크, 모바일 메뉴
5. **상태** — 빈 상태, 로딩 상태, 에러 상태, 가득 찬/오버플로우 상태 확인
6. **콘솔** — 상호작용 후 에러 확인. 새 JS 에러나 실패한 요청 있는가?
7. **반응형** — 해당 시 모바일 및 태블릿 뷰포트 확인
8. **인증 경계** — 로그아웃 시 어떻게 되는가? 다른 사용자 역할은?
